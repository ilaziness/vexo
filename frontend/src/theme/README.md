# 主题配置说明

本文档说明 SSH GUI 客户端的主题配置和颜色方案设计。

终端主题在`terminalThemes.ts`文件，其他文件是应用主题文件配置。

## 设计理念

清爽简洁、专业可靠、护眼舒适

## 颜色方案

### 亮色模式（Light Mode）

**适用场景**：白天使用，光线充足的环境

- **主色调**：`#1976d2` - 清爽的蓝色，代表技术感和专业性
- **次要色**：`#9c27b0` - 紫色系（MUI默认）
- **背景色**：
  - 主背景：`#fafafa` - 柔和的灰色背景，减少眩光
  - 卡片背景：`#ffffff` - 纯白色
- **文本颜色**：
  - 主要文本：`rgba(0, 0, 0, 0.87)` - 高对比度黑色
  - 次要文本：`rgba(0, 0, 0, 0.6)` - 半透明黑色

**特点**：
- 基于MUI v7默认亮色主题，采用偏灰色调
- 减少纯白背景的眩光感
- 蓝色系配色，专业且清爽

### 暗黑模式（Dark Mode）

**适用场景**：夜间使用，弱光环境，长时间工作

- 使用MUI v7默认暗色主题配置
- 标准的Material Design暗色主题
- 适合大多数应用场景

### 蓝色深色模式（Blue Dark Mode）

**适用场景**：夜间使用，弱光环境，长时间工作（原dark配置）

- **主色调**：`#90caf9` - 柔和的浅蓝色，在暗色背景下清晰可见
- **次要色**：`#26c6da` - 青色系
- **背景色**：
  - 主背景：`#0a0e27` - 深蓝黑色，避免纯黑
  - 卡片背景：`#1a1f3a` - 稍浅的深蓝色
- **文本颜色**：
  - 主要文本：`#e3f2fd` - 浅蓝白色，更柔和
  - 次要文本：`rgba(227, 242, 253, 0.7)` - 半透明浅蓝白

**特点**：
- 深蓝黑背景，避免纯黑造成的对比度过高
- 使用浅蓝白色文本，减少刺眼感
- 护眼设计，适合长时间使用

## 语义化颜色

### 成功（Success）
- 亮色：`#2e7d32` / 暗色：`#66bb6a`
- 用于：连接成功、操作完成等

### 错误（Error）
- 亮色：`#d32f2f` / 暗色：`#f44336`
- 用于：连接失败、错误提示等

### 警告（Warning）
- 亮色：`#ed6c02` / 暗色：`#ffa726`
- 用于：警告信息、注意事项等

### 信息（Info）
- 亮色：`#0288d1` / 暗色：`#29b6f6`
- 用于：提示信息、帮助文档等

## 组件样式配置

### 圆角设计
- 按钮、输入框：`8px`
- 卡片：`12px`
- 对话框：`12px`

### 字体
- 系统默认字体栈，优先使用系统原生字体
- 支持中英文、Emoji 显示

### 过渡效果
- 主题切换时有平滑的过渡动画（0.3s）
- 所有交互元素都有适当的 hover 效果

## 使用方式

### 在组件中使用主题颜色

```tsx
import { Box } from "@mui/material";

// 使用 sx prop 访问主题
<Box
  sx={(theme) => ({
    backgroundColor: theme.palette.background.paper,
    color: theme.palette.text.primary,
    borderColor: theme.palette.divider,
  })}
>
  内容
</Box>
```

### 切换主题模式

```tsx
import { useColorScheme } from "@mui/material/styles";

function MyComponent() {
  const { mode, setMode } = useColorScheme();
  
  // 切换到暗黑模式
  setMode('dark');
  
  // 切换到亮色模式
  setMode('light');
  
  // 跟随系统设置
  setMode('system');
}
```

## 自定义扩展

如需修改主题配色，请编辑 `frontend/src/theme/index.ts` 文件。

建议遵循以下原则：
1. 保持亮色和暗色模式的一致性
2. 确保足够的对比度，符合 WCAG 2.1 AA 标准
3. 避免使用过于饱和的颜色
4. 终端应用应优先考虑护眼和长时间使用的舒适度

### 终端CLI模式（Terminal CLI Mode）

**适用场景**：追求经典终端体验，怀旧风格界面

- **主色调**：`#33ff00` - 经典的亮绿色，模仿phosphor monitor
- **次要色**：`#ffb000` - 琥珀色，用于警告和强调
- **背景色**：
  - 主背景：`#0a0a0a` - 深黑背景，支持scanline效果
  - 卡片背景：`#1a1a1a` - 稍浅的深黑
- **文本颜色**：
  - 主要文本：`#33ff00` - 亮绿色文本
  - 次要文本：`#1f521f` - 暗绿色次要文本

**特点**：
- 模仿经典终端的phosphor显示效果
- 高对比度设计，符合终端应用的特点
- 绿色主题配色，营造怀旧的命令行体验
